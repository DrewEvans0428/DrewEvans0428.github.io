Web Applications is a course at the university of South Carolina. It is meant to help prepare students on how to design and build websites!

Assignments

Assignment 1

In assignment 1, we learned the basics of HTML. We learned how to use tables, navigation bars, and make lists. I did my assignment on the top 5 NBA players in the MVP race. This assignment includes links and images as well.

Assignment 2

In assignment 2, we learned the basics of CSS. I decided to do my assignment on my 5 favorite PGA golfers and we used tables once again. We learned how to use background images as well as opacity. We also learned how to properly space items.

Assignment 3

In assignment 3, we learned how to use flexbox. With flexbox we learned how to set up grids and how to align certain items within the flexbox. It was a pretty tough assignment.

Assignment 4

In assignment 4, we were expected to master css and flex layouts. It was a pretty hard and time-consuming assignment.

Project Parts

Project Part 1

In Project Part 1, we decide on our project topic. After deciding the project topic, I decided on what type of data the page will store, manipulate, and what people will view. I decided to do this on my mothers salon.

Project Part 2

In Project Part 2, we make a wireframe on mockitt. Doing that we establish a ground for what our projects structure will look like.
